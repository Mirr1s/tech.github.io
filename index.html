Trabajos

Práctica 1: Boolean Logic

Se busca aprender a utilizar un conjunto de herramientas de software llamado Nand2Tetris y entender algunos elementos de arquitectura de computadores, específicamente lógica booleana y compuertas, los cuales se pueden simular en la suite mencionada. Para su desarrollo, inicialmente se deben entender las partes, funcionalidades y manera de uso del implemento de simulación, también es necesario repasar conceptos de lógica booleana, tales como la conformación de compuertas a partir de chips Nand y los resultados que deben producir cada una.

Ver repositorio en GitHub

Práctica 2:Lógica Aritmética y Lógica Secuencial

Durante esta práctica, se abordaron los conceptos de Aritmética Booleana y Lógica Secuencial. Para llevar a cabo esta actividad, se requiere disponer de los circuitos integrados previamente construidos en la práctica anterior y contar con un entendimiento previo de los temas mencionados.

Ver repositorio en GitHub

Práctica 3:Lenguaje Máquina y Arquitectura de Computadores

En el proyecto 4, nosotros escribimos y ejecutamos algunos programas en lenguaje de máquina de bajo nivel. En particular, escribimos programas en el lenguaje ensamblador Hack, utilizamos un ensamblador para convertirlos en código binario y probamos el código resultante. Dado que la computadora Hack se construiría solo en el siguiente proyecto, en ese proyecto ejecutamos los programas en un emulador de CPU diseñado para ejecutar código binario escrito en el conjunto de instrucciones Hack. En el proyecto 5, integramos los dispositivos ALU y RAM introducidos en los proyectos 2 y 3 en una computadora. Fue un sistema capaz de ejecutar programas escritos en el lenguaje de máquina introducido en el proyecto 4.

Ver repositorio en GitHub

Práctica 4: Ensamblador

Esta práctica se enfoca en el ensamblador, un programa diseñado para traducir el código escrito en un lenguaje de máquina simbólico a un código escrito en un lenguaje de máquina binario. El objetivo consiste en escribir un programa ensamblador.

Ver repositorio en GitHub

Practica 5 - Maquinas virtuales

En la practica 5 se realizaron los proyectos numero 7 y 8 de Nand2Tetris, los cuales se centran en las máquinas virtuales, en un traductor para estas más específicamente. El proyecto 7 se basa en la parte aritmética y de acceso de memoria, mientras que el proyecto 8 se centra en comandos de ramificaciones y funciones.

Ver repositorio en GitHub

Practica 6 - Lenguaje de alto nivel

En la practica 6, hemos creado un programa en Jack, un lenguaje de programación simple con base en objetos y similar a Java. Estos logros sientan las bases para las etapas posteriores de nuestro viaje, donde construiremos un compilador de Jack y un sistema operativo básico.
