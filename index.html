Trabajos

Proyecto 1: Boolean Logic

Se busca aprender a utilizar un conjunto de herramientas de software llamado Nand2Tetris y entender algunos elementos de arquitectura de computadores, específicamente lógica booleana y compuertas, los cuales se pueden simular en la suite mencionada. Para su desarrollo, inicialmente se deben entender las partes, funcionalidades y manera de uso del implemento de simulación, también es necesario repasar conceptos de lógica booleana, tales como la conformación de compuertas a partir de chips Nand y los resultados que deben producir cada una.
